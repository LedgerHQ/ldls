BREAKING_CHANGE(ui-rnative): Migrate Subheader to Composite Pattern API

## Migration Guide

Subheader now uses a composite pattern instead of props.

### Breaking Changes

1. `title` prop removed - use `SubheaderTitle` component
2. `Subheader.Info` and `Subheader.Action` are now separate exports (no dot notation)
3. `SubheaderRow` is now required to wrap title and inline elements
4. `SubheaderInfo` now includes the icon and styling (no longer a wrapper)
5. `SubheaderAction` requires an `onPress` prop and auto-aligns right

### Migration Steps

#### 1. Simple Title Only

**Before:**

```tsx
<Subheader title='Section Title' />
```

**After:**

```tsx
<Subheader>
  <SubheaderRow>
    <SubheaderTitle>Section Title</SubheaderTitle>
  </SubheaderRow>
</Subheader>
```

#### 2. With Info Tooltip

**Before:**

```tsx
<Subheader title='Section Title'>
  <Subheader.Info>
    <Tooltip>
      <TooltipTrigger>
        <Information size={12} />
      </TooltipTrigger>
      <TooltipContent content={<Text>Additional information</Text>} />
    </Tooltip>
  </Subheader.Info>
</Subheader>
```

**After:**

```tsx
<Subheader>
  <SubheaderRow>
    <SubheaderTitle>Section Title</SubheaderTitle>
    <Tooltip>
      <TooltipTrigger>
        <SubheaderInfo />
      </TooltipTrigger>
      <TooltipContent content={<Text>Additional information</Text>} />
    </Tooltip>
  </SubheaderRow>
</Subheader>
```

> **Note:** `SubheaderInfo` now includes the Information icon and interactive styling by default.

#### 3. With Action

**Before:**

```tsx
<Subheader title='Section Title'>
  <Subheader.Action>
    <Link href='/action' appearance='accent' size='sm'>
      Action
    </Link>
  </Subheader.Action>
</Subheader>
```

**After:**

```tsx
<Subheader>
  <SubheaderRow>
    <SubheaderTitle>Section Title</SubheaderTitle>
    <SubheaderAction onPress={handleAction}>
      <Link href='/action' appearance='accent' size='sm'>
        Action
      </Link>
    </SubheaderAction>
  </SubheaderRow>
</Subheader>
```

> **Note:** `SubheaderAction` now uses `onPress` (React Native convention) and automatically aligns to the right.

### New Features

You can now use these additional components:

- `SubheaderCount` - Display numeric count badges
- `SubheaderDescription` - Add text below the title row
- `SubheaderShowMore` - Add a chevron indicator for expandable content
- Interactive rows - Add `onPress` to `SubheaderRow` to make it pressable

### Quick Reference

| Old Pattern                                | New Pattern                                                         |
| ------------------------------------------ | ------------------------------------------------------------------- |
| `<Subheader title="..." />`                | `<SubheaderRow><SubheaderTitle>...</SubheaderTitle></SubheaderRow>` |
| `<Subheader.Info>...</Subheader.Info>`     | `<SubheaderInfo />` (with Tooltip wrapper)                          |
| `<Subheader.Action>...</Subheader.Action>` | `<SubheaderAction onPress={...}>...</SubheaderAction>`              |
| `onClick`                                  | `onPress` (React Native convention)                                 |

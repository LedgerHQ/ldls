BREAKING_CHANGE(ui-react): Migrate Subheader to Composite Pattern API

## Migration Guide

Subheader now uses a composite pattern instead of props.

### Breaking Changes

1. `title` prop removed - use `SubheaderTitle` component
2. `Subheader.Info` and `Subheader.Action` are now separate exports (no dot notation)
3. `SubheaderRow` is now required to wrap title and inline elements
4. `SubheaderInfo` now includes the icon and styling (no longer a wrapper)
5. `SubheaderAction` requires an `onClick` prop and auto-aligns right

### Migration Steps

#### 1. Simple Title Only

**Before:**

```tsx
<Subheader title='Section Title' />
```

**After:**

```tsx
<Subheader>
  <SubheaderRow>
    <SubheaderTitle>Section Title</SubheaderTitle>
  </SubheaderRow>
</Subheader>
```

#### 2. With Info Tooltip

**Before:**

```tsx
<Subheader title='Section Title'>
  <Subheader.Info>
    <Tooltip>
      <TooltipTrigger asChild>
        <Information size={12} className='shrink-0 text-muted' aria-label='More information' />
      </TooltipTrigger>
      <TooltipContent>Additional information</TooltipContent>
    </Tooltip>
  </Subheader.Info>
</Subheader>
```

**After:**

```tsx
<Subheader>
  <SubheaderRow>
    <SubheaderTitle>Section Title</SubheaderTitle>
    <Tooltip>
      <TooltipTrigger>
        <SubheaderInfo />
      </TooltipTrigger>
      <TooltipContent>Additional information</TooltipContent>
    </Tooltip>
  </SubheaderRow>
</Subheader>
```

> **Note:** `SubheaderInfo` now includes the Information icon and interactive styling by default.

#### 3. With Action

**Before:**

```tsx
<Subheader title='Section Title'>
  <Subheader.Action>
    <Link href='/action' appearance='accent' size='sm'>
      Action
    </Link>
  </Subheader.Action>
</Subheader>
```

**After:**

```tsx
<Subheader>
  <SubheaderRow>
    <SubheaderTitle>Section Title</SubheaderTitle>
    <SubheaderAction onClick={handleAction}>
      <Link href='/action' appearance='accent' size='sm'>
        Action
      </Link>
    </SubheaderAction>
  </SubheaderRow>
</Subheader>
```

> **Note:** `SubheaderAction` now requires an `onClick` prop and automatically aligns to the right.

### New Features

You can now use these additional components:

- `SubheaderCount` - Display numeric count badges
- `SubheaderDescription` - Add text below the title row
- `SubheaderShowMore` - Add a chevron indicator for expandable content
- Interactive rows - Add `onClick` to `SubheaderRow` to make it clickable

### Quick Reference

| Old Pattern                                | New Pattern                                                         |
| ------------------------------------------ | ------------------------------------------------------------------- |
| `<Subheader title="..." />`                | `<SubheaderRow><SubheaderTitle>...</SubheaderTitle></SubheaderRow>` |
| `<Subheader.Info>...</Subheader.Info>`     | `<SubheaderInfo />` (with Tooltip wrapper)                          |
| `<Subheader.Action>...</Subheader.Action>` | `<SubheaderAction onClick={...}>...</SubheaderAction>`              |

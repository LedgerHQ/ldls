## Introduction

Tile components are compact, card-like interactive elements designed for horizontal lists, carousels, or grids. They feature a vertical layout with a required spot icon and title, as well as an optional description, tag, and a secondary action button visible on hover.

> View in [Figma](https://www.figma.com/design/JxaLVMTWirCpU0rsbZ30k7/2.-Components-Library?node-id=5783-1328).

## Anatomy

<Canvas of={TileStories.Base} />

- **Leading Content**: Visual icon or element at the top
- **Title**: The main label of the item (required)
- **Description (optional)**: Provides additional context
- **Trailing Content (optional)**: Trailing custom content at the bottom of the tile
- **Secondary Action (optional)**: Action button in the top right corner, visible on hover

### Overview

### Appearance

Two appearance variants are available:

- **no-background** (default): Transparent background with hover/pressed states
- **card**: Surface background color with hover/pressed states

<Canvas of={TileStories.AppearanceShowcase} />

### States

Tiles support multiple interactive states:

- **Default**: Base state with no interaction
- **Hover**: Background color changes on hover
- **Pressed**: Darker background when clicked
- **Focused**: Focus ring appears on keyboard navigation
- **Disabled**: Non-interactive state with disabled styles

<Canvas of={TileStories.StateShowcase} />

### Disabled State

When disabled, tiles become non-interactive and display with disabled styles.

<Canvas of={TileStories.DisabledShowcase} />

### Spot Variants

Tile components support various spot icons to categorize or visualize the content, including coin icons.

## Basic Usage

```tsx
import { Tile, Spot } from '@ledgerhq/ldls-ui-react';
import { Settings } from '@ledgerhq/ldls-ui-react/symbols';

function MyComponent() {
  return (
    <Tile
      title='Settings'
      leadingContent={<Spot appearance='icon' icon={Settings} />}
      onClick={() => console.log('Clicked!')}
      appearance='no-background'
    />
  );
}
```

### Appearance Variants

```tsx
// Transparent background (default)
<Tile
  title='Settings'
  leadingContent={<Spot appearance='icon' icon={Settings} />}
  appearance='no-background'
/>

// Card/Surface background
<Tile
  title='Settings'
  leadingContent={<Spot appearance='icon' icon={Settings} />}
  appearance='card'
/>
```

### Disabled State

```tsx
<Tile
  title='Settings'
  description='Unavailable'
  leadingContent={<Spot appearance='icon' icon={Settings} disabled />}
  disabled
/>
```

### With Coin Icon

```tsx
import { Tile, Spot } from '@ledgerhq/ldls-ui-react';

function MyComponent() {
  return (
    <Tile
      title='Bitcoin'
      description='BTC'
      leadingContent={<Spot appearance='coin' icon='btc' />}
      onClick={() => console.log('Clicked!')}
    />
  );
}
```

### Horizontal List

```tsx
<div className='flex overflow-x-auto'>
  {items.map((item, i) => (
    <Tile
      key={item.id}
      title={item.title}
      description={item.description}
      leadingContent={<Spot appearance='icon' icon={item.icon} />}
    />
  ))}
</div>
```

# üìè Size

Width and height define the fixed dimensions of elements, controlling their span and contributing to layout consistency.

## ‚ö° Structure

All size tokens follow the structure: `size-{size}` where the size value represents **pixels**.

<img src={sizeTaxonomy} alt="Size token taxonomy structure" />

Example: `size-44`

<Markdown>
  {`
| Element | Definition | Example |
| ------- | ---------- | ------- |
| size | Defines fixed dimensions of element | size |
| Size | Defines the value in pixels | 0, 1, 2, 4, 6, 8, 10, 12, 14.. |
`}
</Markdown>

## üìê Size Scale

Size tokens provide a consistent set of dimensions that can be used for both width and height properties. These tokens ensure that elements maintain proportional and harmonious relationships across the interface.

### Usage

Size tokens can be applied in three ways:

- As a square dimension using the `size-*` class
- As a width using the `w-*` class
- As a height using the `h-*` class

<Markdown>
{`
\`\`\`jsx
// Square element (both width and height)
<div className="size-32" />

// Width only

<div className="w-32" />

// Height only

<div className="h-32" />
\`\`\`
`}
</Markdown>

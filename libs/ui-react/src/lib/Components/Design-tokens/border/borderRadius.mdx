# üî≤ Border Radius

Border radius defines the curvature of UI elements, ensuring consistency and visual harmony across our products.

## ‚ö° Structure

All radius tokens follow the structure: `border-radius-size`

Example:

- `border-radius-md`
- `border-radius-lg`

**üé® For designers:** The table below lists our border-radius tokens, showing their typical usage to help maintain visual consistency across the interface.

<Markdown>
  {`
| Size | Usage |
| ---- | ----- |
| none | no rounding; for sharp, rectangular corners |
| xs   | very slight rounding; suitable for compact tags or checkbox |
| sm   | light rounding on inputs, search, or small components |
| md   | standard rounding on small components like segmented control |
| lg   | standard rounding on larger surfaces like feature cards or containers |
| xl   | soft corner for larger surfaces |
| 2xl  | extra-soft corner for larger surfaces |
| full | fully rounded (pill shape); used for all buttons, chips and circular elements |
`}
</Markdown>

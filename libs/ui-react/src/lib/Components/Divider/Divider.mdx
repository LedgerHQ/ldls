## Introduction

The Divider component provides a simple, semantic way to visually separate content sections in your interface. It creates a horizontal or vertical line that helps organize information and improve content hierarchy.

### Orientation

Dividers support both horizontal and vertical orientations to accommodate different layout needs:

- `horizontal` (default): Creates a horizontal line spanning the full width of its container
- `vertical`: Creates a vertical line with a height determined by its container or custom styling

## Responsive Layout

Dividers automatically adapt to their container's dimensions. For horizontal dividers, they span the full width. For vertical dividers, set an explicit height using the `className` prop.

## Accessibility

The Divider component includes proper accessibility attributes:

- Uses `role="separator"` for semantic meaning
- Includes `aria-orientation` to indicate direction
- Works with screen readers to convey content structure

## Basic Usage

```tsx
import { Divider } from '@ledgerhq/lumen-ui-react';

function MyComponent() {
  return (
    <div>
      <p>Content above the divider</p>
      <Divider />
      <p>Content below the divider</p>
    </div>
  );
}
```

### Vertical Divider

For vertical separation between content:

```tsx
import { Divider } from '@ledgerhq/lumen-ui-react';

function MyComponent() {
  return (
    <div className='flex items-center gap-16'>
      <span>Left content</span>
      <Divider orientation='vertical' className='h-48' />
      <span>Right content</span>
    </div>
  );
}
```

Note: For vertical dividers, you typically need to set an explicit height using the `className` prop.

### In List Components

Dividers are perfect for separating items in lists, menus, or cards:

```tsx
import { Divider } from '@ledgerhq/lumen-ui-react';

function MyList() {
  return (
    <div className='flex flex-col rounded-lg border border-muted'>
      <div className='p-16'>Item 1</div>
      <Divider />
      <div className='p-16'>Item 2</div>
      <Divider />
      <div className='p-16'>Item 3</div>
    </div>
  );
}
```

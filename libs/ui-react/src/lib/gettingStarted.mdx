# üëã Welcome to LDLS

The Ledger Design Language System (LDLS) is a comprehensive design system that provides a unified set of principles, components, and tools for building consistent Ledger applications.

## üéØ Core Principles

- **Consistency**: Unified design language across all Ledger products
- **Efficiency**: Focus on what matters - use proven solutions for common patterns
- **Speed**: Fast development with ready-to-use components and utilities
- **Flexibility**: Adaptable components that work across different contexts
- **Accessibility**: Built with accessibility in mind from the ground up

### 2. Components

Our components are built with:

- React + TypeScript
- Tailwind CSS for styling
- Storybook for documentation
- Comprehensive test coverage

### 3. Development Setup

#### Required Tools

- Node.js 18+
- npm or yarn
- React 18+

**Quick Start**

1 . Install the package:

```bash
npm install @ldls/ui-react
```

2 . Configure Tailwind:
See our [Tailwind Setup Guide](/docs/tailwind-setup--docs)

3 . Start using components:

```tsx
import { Button } from '@ldls/ui-react';

function App() {
  return (
    <Button appearance="base">
      Get Started
    </Button>
  );
}
```

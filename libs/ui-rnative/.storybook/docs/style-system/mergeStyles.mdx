# mergeStyles

The `mergeStyles` function is a utility for merging multiple style objects.

```tsx
import { mergeStyles } from '@ledgerhq/lumen-ui-rnative/styles';

const style = mergeStyles({ padding: 's16' }, { margin: 's8' });
```

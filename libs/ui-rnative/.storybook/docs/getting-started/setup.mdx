## Installation
Install the packages and their peer dependencies:

```bash
# Install the UI Kit
npm install @ledgerhq/lumen-ui-rnative @ledgerhq/lumen-design-core

# Install peer dependencies
npm install react-native@^0.77.0 react@^18.3.1
```
